# Gun Bunny

## What is it?

Gun Bunny is a Contra like game with the ability to aim in any direction through the use of the right stick.  This game was made for Global Game Jam 2023.

## Controls

Move: D-Pad
Jump: L Button
Shoot: Right Stick in direction you want to shoot
